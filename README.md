# Pinella
Segnapunti per Burraco con avatar, tornei a girone all’italiana e gestione roster. PWA statica, hosted su GitHub Pages.

## Struttura

- `index.html` — markup dell'app
- `assets/styles.css` — stili
- `assets/js/main.js` — logica
- `assets/img/` — icone (manifest + apple-touch)
- `manifest.webmanifest` — manifest PWA
